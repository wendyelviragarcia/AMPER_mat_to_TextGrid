# AMPER `.mat` to Praat TextGrid Converter

This repository contains an **R script** that converts `.mat` files generated by **AMPER projects** (using the **amper06** Matlab software) into **Praat TextGrid files** with aligned intervals.

If the written corpus is available in a `.txt` file, the script can also insert the **orthographic sentence and its IPA transcription** aligned with the intervals in the resulting TextGrid.

## Overview

The AMPER tools export prosodic annotation data in MATLAB `.mat` files. These files contain temporal information about segment clusters (C V) that can be used to generate aligned annotations in Praat.

This script reads those `.mat` files and:

1. Extracts the timing information from the AMPER output.

2. Creates a **Praat TextGrid** with the corresponding aligned intervals.

3. Optionally inserts:
   
   * the **orthographic sentence**
   * the **IPA transcription**

Both are aligned with the intervals if the corpus transcription file is provided else you will have a C V annotation.

## Requirements

The script requires the following R packages:

```r
R.matlab
rPraat
stringr
```

You can install them running this in your RStudio Console:

```r
install.packages(c("R.matlab", "rPraat", "stringr"))
```

## Input Files

### 1. AMPER `.mat` files

These files are generated by **amper06** within AMPER workflows and contain the segmentation and timing information used to create the TextGrid intervals.

### 2. Corpus transcription file (optional)

A tab-delimited `.txt` file containing the corpus sentences and their IPA transcription. If you are part of AMPER project this was a file that we used to send to Grenoble or Torino.

Example structure:

| code      | orthography                            | IPA                                          |
| --------- | -------------------------------------- | -------------------------------------------- |
| xp01bwka1 | La cítara magrebí se toca con obsesión | la ˈƟitaɾa maɣ ̞ɾeˈβ̞i se ˈtoka kon oseˈsjon |

Fields:

* **code** – identifier matching the `.mat` filename
* **orthography** – written sentence
* **IPA** – phonetic transcription

If this file is not provided, the script will still create the TextGrid intervals but with C V labels.

## Output

For each `.mat` file, the script produces a **Praat TextGrid** containing:

* tier 1 (phrase): orthographic text or phrase code
* tier 2 (intervals): interval boundaries derived from the AMPER `.mat` file: optional IPA transcription

## Notes

* The `.mat` files must come from **AMPER software (amper06)**.
* The corpus `.txt` file must use the same **codes as the `.mat` filenames** to allow matching.
